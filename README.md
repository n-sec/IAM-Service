## Requirements
 - NodeJS

 - MongoDB

## Installation
 1. Clone this repository:
 ```
 $ git clone https://github.com/n-sec/IAM-Service.git
 ```
 2. Enter the directory:
 ```
 $ cd IAM-Service
 ```
 3. Install and update the packages:
 ```
 $ npm install
 ```
 2. Add the .env file to the project's root directory:
 ```
 $ cp env .env
 ```
